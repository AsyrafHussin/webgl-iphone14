# 3D Iphone 14 Pro Max

## Getting Started

```bash
# Clone the repo
git clone https://github.com/AsyrafHussin/webgl-iphone14.git

# Navigate to project folder
cd webgl-iphone14

# Initial Setup
npm install

# Serve the application on http://127.0.0.1:3000
npm run dev
```
